## 2. TTS — Синтез речи (своя модель)

### Варианты TTS моделей

| Модель | VRAM | Качество | Скорость | Лицензия |
|---|---|---|---|---|
| **Silero TTS** | ~1 GB CPU | Среднее | Быстрая | Apache 2.0 |
| **Coqui XTTS v2** | ~3 GB GPU | Высокое | Средняя | CPML |
| **Kokoro TTS** | ~2.5 GB GPU | Высокое | Быстрая | Apache 2.0 |
| **Piper TTS** | ~0.5 GB CPU | Среднее | Быстрая | MIT |
| **Своя модель** | ? | ? | ? | Своя |

### Расчёт стоимости TTS на примере Kokoro/Silero

**Параметры:**
- Средний звонок: 3 минуты
- TTS генерирует ~150 слов/мин = 450 слов/звонок
- На 1 звонок: ~30 секунд генерации на GPU
- При 200 звонках/мес (тариф Профессиональный): 100 минут генерации/мес

**Вариант A: TTS на том же сервере (RTX 3060)**
- Дополнительная VRAM: ~2.5 GB (Kokoro)
- Итого VRAM: 4 GB (LLM) + 2.5 GB (TTS) = 6.5 GB
- RTX 3060 12GB — **хватит с запасом**
- **Дополнительная стоимость: 0 ₽** (на том же сервере)

**Вариант B: TTS на CPU (Silero/Piper)**
- VRAM: 0 GB (работает на CPU)
- CPU нагрузка: ~10-20% на 1 ядро
- **Дополнительная стоимость: 0 ₽** (на том же сервере)

**Вариант C: Отдельный TTS сервер**
- Минимальный GPU: RTX 3050 8GB (~8 000 ₽/мес)
- Или CPU-сервер: ~3 000 ₽/мес (для Silero/Piper)

### Рекомендация: TTS на том же сервере
- Kokoro TTS на GPU или Silero на CPU
- **Дополнительная стоимость: 0 ₽**
- Один сервер RTX 3060: **12 000 ₽/мес** — и LLM, и TTS

## 5. Сравнение: свой сервер vs API

### Свой сервер (Qwen 7B INT4)

| Метрика | Значение |
|---|---|
| Стоимость/мес | 12 700 ₽ |
| Звонков/мес | ~1 500 |
| Стоимость/звонок | **~8.5 ₽** |
| Контроль данных | Полный |
| Задержка | 2-3 сек |
| Масштабируемость | Линейная (+сервер) |

### API (OpenAI GPT-4o mini)

| Метрика | Значение |
|---|---|
| Вход: ~0.15 $/1M токенов | ~1.5 ₽/звонок |
| Выход: ~0.60 $/1M токенов | ~6 ₽/звонок |
| Стоимость/звонок | **~7.5 ₽** |
| Контроль данных | Нет |
| Задержка | 1-2 сек |
| Масштабируемость | Автоматическая |

### API (Yandex GPT)

| Метрика | Значение |
|---|---|
| Вход: ~0.90 ₽/1K токенов | ~3 ₽/звонок |
| Выход: ~1.50 ₽/1K токенов | ~5 ₽/звонок |
| Стоимость/звонок | **~8 ₽** |
| Контроль данных | Частичный |
| Задержка | 1-2 сек |
| Масштабируемость | Автоматическая |

### Вывод

| Вариант | Стоимость/звонок | Плюсы | Минусы |
|---|---|---|---|
| **Свой сервер** | ~8.5 ₽ | Полный контроль, данные в РФ | Нужно масштабировать вручную |
| **OpenAI API** | ~7.5 ₽ | Дешевле, масштабируется | Данные в США, нет контроля |
| **Yandex GPT** | ~8 ₽ | Данные в РФ, масштабируется | Дороже при росте, зависимость |

**Рекомендация:** Начать с API (Yandex GPT) для быстрого запуска, параллельно разворачивать свой сервер на Qwen 7B. При 100+ клиентах свой сервер становится выгоднее.
